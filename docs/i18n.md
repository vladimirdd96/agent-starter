## Rules

1. Never hardcode user-visible strings — always go through the translation hook/function
2. Add all keys to **all** supported locales in the same commit
3. Use `{{paramName}}` placeholders for interpolated values — match names across locales
4. Plural forms use separate keys (e.g. `item` singular, `items` plural)
